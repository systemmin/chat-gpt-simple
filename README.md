# 简易版Chat-GPT 

简易版chat-gpt，技术实现node+html+css+js，text/event-stream流接入，便于大家理解底层原理。



- `node-fetch` 向`chat api 发起请求
- 处理 `text/event-stream`
- http 请求进行转发 页面
- 页面接收响应，处理数据，md转html，html回显页面
